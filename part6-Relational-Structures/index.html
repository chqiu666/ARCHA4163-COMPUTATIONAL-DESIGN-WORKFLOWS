Multiverse Collapse

This demo uses a tiny model 'distilgpt2' with the transformerjs library to run text prediction generation on the website. The NLP text processing is between the complex data input for real-world crime predictions and the limited computational experiment of a gerbil inside a glass enclosure. Through this branching tree, the demo showcases how machine predictions, driven by probabilities, can collapse into the freedom of individual choice.

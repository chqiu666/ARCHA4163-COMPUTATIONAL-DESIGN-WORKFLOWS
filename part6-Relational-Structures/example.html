Digital Object 3: Relational Structures

This section explores relational data visualization using D3.js, demonstrating how to create interactive network graphs and relationship visualizations that help understand connections and patterns in complex data structures.

D3.js Foundation: Simple Force-Directed Graph

A basic D3.js example demonstrating core force simulation concepts. This simple network graph shows basic connections between nodes with drag interactions and force-directed layout. Features include basic node and link visualization with automatic positioning.

D3.js Advanced: Interactive Network Graph with Enhanced Features

An advanced D3.js example demonstrating enhanced features including custom data properties, relationship types, interactive tooltips, and dynamic visual encoding. This network graph shows relationships between people with different roles, departments, and connection strengths. Features include drag interactions, hover effects with tooltips, and dynamic visual encoding based on data properties.

D3.js CSV Integration: Network Graph with External Data Sources

A demonstration of loading network data from external CSV files, showing how to separate data from visualization logic. This example loads node and edge data from separate CSV files, processes the data types appropriately, and creates the same interactive network visualization. This approach enables easy data updates without modifying code and supports larger datasets. Note: This example requires running the HTML file in a local server due to browser security restrictions when loading external CSV files. If you're unsure how to set up a local server, please refer to the "Running a Local Server Tutorial" in the course materials.
